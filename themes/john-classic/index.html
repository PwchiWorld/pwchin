Hmmm John theme ...

Fairly simple, no fuss about it, suitable for those who enjoy simplicity but put their image on the spotlights.

About John

It is a mobile-friendly premium theme. Uses subtle shadow effects and high contrast on interactive UI, emmm, yes, the buttons.

About the layout

  • Top Badge.  Indicates the class of the user. Read more about user-roles on FAQ page
  • Full header. Supports wide, full header image. When enabled, the "pill" is displayed. Username with a star as prefix which is, an award on its own.
  • Profile Picture. Avatar-like image in the center of the profile page.
  • Name of the profile can be different than your username so, choose wisely.
  • About section to hold added content
  • Social buttons are optional but recommended

How to use this theme?

  1. Go to Create Pwchin Profile page.
  2. Choose a name for this profile, can be similar with username or a different title, it must be unique.
  3. From drop-down, choose the  John option.
  4. Add content to About section.
  5. Optional - choose Profile Picture, the avatar. It will display with round corners.
  6. Optional - choose header image. Opt for landscape (wider) type rather than portrait. Works best with 1000px width per 450px height images. 
  7. Optional - social links are a good way to soft advertise your social accounts.
  8. Important - Privacy option. Can set the profile to Public or Private.
