About Neo n

It is a mobile-friendly premium theme. Uses subtle animated touches for the most keen eyes.

It is already glowing neon style so more visible animations would throw users on a journey they have not expected. It is safe, don't worry.

Let's dive into the techy stuff

  • Top "Pill" badge is user's role. Indicates the class of the user. Read more about user-roles on FAQ page
  • Supports wide, full header image. When enabled, the "pill" is displayed. Username with a star as prefix which is, an award on its own.
  • Profile Picture.  Avatar-like image in the center of the profile page.
  • Name of the profile can be different than your username so, choose wisely.
  • About section to hold added content
  • Social buttons are optional but recommended
